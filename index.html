Projects

Funky Game in Java

Project Description: This Java code was created in one of my past programming courses. The purpose was to make a fully functioning game with pieces on the board that jump to random places and get rid of one another if they land in the same spot. This process loops endlessly until there is one piece left on the board to be declared the winner.

The five classes that went into the creation of this game each did different things like creating the board, randomizing where the piece is placed, and moving said token around. Though this project was extremely challenging to get correct, I came out of it with so much new knowledge on how subclasses can work together to produce something new.

Rocket Animation in DrRacket

Project Description: This project was made just a few weeks ago for my CSCI-C211 course. Whenever we are instructed to create a function, in this case to land a rocket, we follow an algoritm known as the design recipe. This six-step process includes data definitions, signature, function examples, function template, function definition, and testing.

Though it is a bit tedious to repeat this process over and over, I have learned how to be more methodical with what information is going into my programs and how it intertwines with each other. The projects in this course have challenged me to channel my thoughts in a different way.
